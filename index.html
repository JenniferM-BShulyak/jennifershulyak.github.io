About Me

My Story

After five years in education, I began the Data Analytics and Visualizations Boot Camp at the University of Texas at Austin so that I could follow my old and new passions of learning and coding through a career in data analytics. As a high school math educator I was devoted to improving the quality of curriculum and instruction using data, whether it be data I had collected myself or based on expert research. The COVID pandemic only emphasized the importance of quality online math curriculum instruction and resources, which led me to nurture my growing interest in coding and data science. I taught myself how to create interactive Desmos websites using its Computation Layer, to curate my teaching website with HTML and CSS, and eventually to learn Javascript and Python all without little or no direction. These experiences ignited a desire to learn more about data science and to pursue a new career. .

The Austin Housing Market: Python, Javascript, Tableau

In this project, I worked with a team to extract Zillow zestimate (estimated listing) and rental data for the Austin metro. We worked to build a temporary webpage with interactive visualization and tables using Javascript, HTML, and Plotly. Data was cleaned and transformed in Python and stored in a PostegreSQL database. My roles were to extract and trasform the data as well as to create the line plots looking at the zestimate data over time in Plotly. I worked closely with other team members to build the JS endpoints and connect with the PostegreSQ database using SQLAlchemy.

I later decided to extend the project by created a Tableau Dashboard to better summarize trends and compare capabilities with Plotly.

NLP Model and data cleaning in Python for Crypto Data

Are cryptocurrencies an attractive investment given current economic indicators and sentiment? To answer this question, data was extracted from multiple sources and transformed in Python. The machine learning models were built to analyze sentiment and forecast market performance. In order to present this analysis to clients, I would provide them with Tableau Dashboards displaying the results.

  • View Project

Some Key Findings:

    1. Bitcoin sentiment fluctuates with world events and there is some correlation with Bitcoin news sentiment and it market performance.
    2. Vader more accurately predicted what the team identified as being positive headlines while the Random Forest NLP labeled them as neutral. Accuracy for both models varies wildly: 40-80%.

Skills/tools used: Python, Pandas, PostgresSQL, PGAdmin4, Vader, Random Forest Model (AI), NewsAPI
